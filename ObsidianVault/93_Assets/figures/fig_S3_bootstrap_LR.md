# Figure S3 — Bootstrap Null Distribution of JMN LR Statistic

**LaTeX label (canonical):** `ch1:fig:S3_bootstrap_LR`
**Caption:** "Bootstrap null distribution of the Johansen, Mosconi & Nielsen (2000) LR statistic, H₀: β_{e*} = 0."
**File:** `Chapter1/figures/s3_bootstrap_LR_dist.pdf`

## What it shows

Bootstrap null distribution for the JMN (2000) likelihood-ratio test of the restriction that the exploitation-rate coefficient in the cointegrating vector is zero (β_{e*} = 0). The observed test statistic is marked against the bootstrap distribution.
