# Equation — Capacity Benchmark Normalization

**LaTeX label (canonical):** `ch1:eq:capacity_benchmark`
**Form:** `y* = y / e`  (or equivalent normalization)

Defines the capacity benchmark: productive output normalized by the exploitation rate. This normalization is Shaikh's key step that separates the trend component from the cyclical reserve-army fluctuation.
