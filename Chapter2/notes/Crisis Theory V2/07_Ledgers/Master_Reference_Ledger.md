# Master Reference Ledger

Use this as the **master control table** for sources.

| Key | Author | Year | Title / Short title | Type | School / tradition | Main concept | Chile relevance | Function in argument | Status | Linked note |
|---|---:|---|---|---|---|---|---|---|---|---|
| REF-001 |  |  |  | book/article/chapter |  |  | high/medium/low | anchor / contrast / evidence / definition / historiography / method | unread / reading / processed | [[ ]] |

## Function field guide
- **anchor** → indispensable source around which an argument is built
- **contrast** → source used to sharpen disagreement
- **evidence** → empirical or historical support
- **definition** → concept clarification
- **historiography** → situates a debate in time
- **method** → guides how to read the case or build the analysis

## Suggested workflow
1. Add source.
2. Assign function.
3. Create a linked note in `03_Authors_and_Schools/` or `09_Workbench/`.
4. Backlink from concept or case notes.
