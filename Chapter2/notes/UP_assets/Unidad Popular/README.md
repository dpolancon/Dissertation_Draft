# Artefacto reutilizable — Unidad Popular (1957–1973)

Este paquete reorganiza el apunte **“El auge y caída de la Unidad Popular en Chile (1957–1973): Una mirada de larga duración”** como un artefacto de Markdown reutilizable para docencia, historiografía, diseño de clases, redacción de capítulos y futuras expansiones.

## Qué incluye

- `00_PLAN_DE_ARTIFACTO.md` — lógica general del sistema y plan de reutilización.
- `01_NOTA_MAESTRA.md` — versión central y navegable del argumento.
- `02_MAPA_ARGUMENTAL.md` — reverse outline / mapa de tesis, mecanismo, evidencia y fuentes.
- `03_LEDGER_DE_FUENTES.md` — registro funcional de fuentes por rol analítico.
- `04_BIBLIOGRAFIA_Y_AUDITORIA.md` — bibliografía depurada + auditoría de referencias.
- `05_REGISTRO_DE_TABLAS_Y_FIGURAS.md` — inventario de objetos empíricos y su función.
- `06_PROXIMOS_PASOS.md` — plan de expansión del artefacto.
- `source/apunte_HistoriaSocial_19571973.pdf` — PDF fuente original.
- `assets/README.md` — guía para poblar activos visuales si luego quieres exportar figuras.

## Ruta recomendada de uso

1. Lee `01_NOTA_MAESTRA.md` para trabajar el texto principal.
2. Usa `02_MAPA_ARGUMENTAL.md` cuando necesites reestructurar, resumir o convertir a slides.
3. Usa `03_LEDGER_DE_FUENTES.md` y `04_BIBLIOGRAFIA_Y_AUDITORIA.md` para conservar trazabilidad bibliográfica.
4. Usa `05_REGISTRO_DE_TABLAS_Y_FIGURAS.md` si vas a llevar esto a clase, paper, o presentación.
5. Usa `06_PROXIMOS_PASOS.md` como backlog editorial.

## Principio de diseño

El artefacto separa:
- argumento principal,
- arquitectura del argumento,
- fuentes,
- referencias,
- objetos empíricos,
- y agenda de expansión.

Así evitas que la bibliografía quede enterrada al final de un texto único y conviertes el apunte en un pequeño sistema de investigación-docencia.
