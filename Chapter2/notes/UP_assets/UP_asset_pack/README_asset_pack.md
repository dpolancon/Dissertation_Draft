# Asset Pack for "Revisiting the Political Economy of the Rise and Fall of the Unidad Popular"

This folder contains a set of complementary assets built around the restructured presentation notebook.

## What is included

### 1. `01_Abstracts_and_Pitch_Versions.md`
A set of compact presentation-ready formulations of the project:
- 75-word pitch
- 150-word abstract
- 250-word conference abstract
- 1-sentence thesis statement
- 3-sentence version for proposals and email outreach

### 2. `02_Talk_Script_12_Minutes.md`
A speaking script for a short conference presentation. It follows a clear arc:
problem, argument, framework, periodization, and takeaway.

### 3. `03_Argument_Evidence_Matrix.md`
A claim-by-claim matrix connecting the main argument to the types of evidence needed. It distinguishes between:
- conceptual claims
- historical claims
- empirical/macroeconomic claims
- claims that still require stronger sourcing

### 4. `04_Periodization_Control_Sheet.md`
A tighter periodization aid that standardizes each phase around:
- accumulation pattern
- crisis tendency
- class dynamics
- state form / institutional adjustment
- transition mechanism

### 5. `05_Paper_Conversion_Blueprint.md`
A roadmap for converting the presentation into a paper or chapter draft. It includes:
- suggested section order
- what to cut
- what to expand
- what to move into notes or appendices

### 6. `06_Handout_One_Page.md`
A compact conference handout version of the project, designed for circulation before or after a talk.

## Suggested order of use

If the goal is to present soon:
1. start with `01_Abstracts_and_Pitch_Versions.md`
2. rehearse with `02_Talk_Script_12_Minutes.md`
3. share `06_Handout_One_Page.md`

If the goal is to write the paper:
1. start with `05_Paper_Conversion_Blueprint.md`
2. control argument drift with `03_Argument_Evidence_Matrix.md`
3. stabilize chronology with `04_Periodization_Control_Sheet.md`

## Core editorial principle

The presentation works best when the argument is kept visible at all times:

> The rise and fall of the Unidad Popular is explained through the articulation of a twin crisis: the crisis of the Prebischian Populist National State and the crisis of global Fordism.

Everything else in the pack is designed to keep that sentence from getting buried under chronology, detail, or historiographic overload.
