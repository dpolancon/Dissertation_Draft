# Industrial Patronage Reference Master Ledger

This ledger reorganizes the essay's references by the job they perform. Its purpose is not only bibliographic control. It is argumentative control.

## Function legend

- **Anchor**: defines the conceptual frame or major category.
- **Mechanism**: explains how the process works.
- **Context**: supplies historical or comparative background.
- **Evidence**: provides empirical or case support.
- **Positioning**: helps situate your intervention relative to a literature.
- **Verify/replace**: should not carry major argumentative weight until bibliographic cleanup is complete.

## J. How to use this ledger while drafting

For each paragraph you draft, ask:
1. What job is this citation doing?
2. Could a stronger source do that job better?
3. Am I citing context when I actually need a mechanism text?
4. Am I using a weak item where an anchor or empirical source should stand?

If a citation has no job, cut it.
