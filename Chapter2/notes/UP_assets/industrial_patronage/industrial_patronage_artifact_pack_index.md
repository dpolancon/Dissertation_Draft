# Industrial Patronage Markdown Artifact Pack

This package contains the converted paper and two reusable dissertation-writing artifacts.

## Files

1. [Essay in Markdown](industrial_patronage_english_apa.md)
2. [Dissertation Reframing Workbench](industrial_patronage_dissertation_reframing_workbench.md)
3. [Master Reference Ledger](industrial_patronage_reference_master_ledger.md)

## Suggested use order

1. Read the essay Markdown file to recover the prose.
2. Use the workbench to redesign the chapter around a sharper analytical object.
3. Use the reference ledger to decide which sources are anchors, which are empirical supports, and which need verification or replacement.
